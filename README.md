# Space Drag

Terminal-only OpenVR space-drag prototype in Python.

## Start

```bash
python3 -m pip install -r requirements.txt
python3 space_drag.py
```

Make sure SteamVR is already running before you start the script.

## Controls

- Hold right joystick click to drag the playspace.
- Press left joystick click to toggle gravity.
- Press `R` in the terminal to recalibrate height and floor.
- Press `Q` in the terminal to quit.

## Files

- `space_drag.py`: main runtime.
- `requirements.txt`: Python dependency list.
